# Event Registration Management System

## Description
This is a C# Console Application that manages event registrations using file handling only.

## Features
- Add Record
- View Records
- Search Records
- Update Records
- Soft Delete
- Hard Delete
- Audit Logging
- Report Generation
- Persistent Storage
- Validation
- Checksum Generation

## Technologies Used
- C#
- .NET 10
- File Handling
- Visual Studio

## Storage Files
- registrations.txt
- audit_log.txt

## How To Run
1. Open project in Visual Studio
2. Press CTRL + F5
